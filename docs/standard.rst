Standard header adaptors
========================

There are fourteen standard adaptors. Two `delete headers`_,
there are eight adaptors that `add headers`_, and three complex
adaptors that `modify headers`_.

Delete headers
--------------

Both the ``Disposition-Notification-To`` and
``Return-Receipt-To`` headers are deleted.
